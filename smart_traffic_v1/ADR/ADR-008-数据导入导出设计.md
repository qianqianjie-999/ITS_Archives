# ADR-008: 数据导入导出设计

## 状态
已接受

## 背景
系统需要支持批量数据的导入和导出功能，方便数据迁移、数据备份和数据分析。数据量预估在万级记录以内，Excel是最常用的数据交换格式。

## 决策
采用 openpyxl 库处理 Excel 文件，设计统一的导入导出服务。

### 技术选型

| 组件 | 用途 |
|------|------|
| openpyxl | Excel读写（xlsx格式） |
| ExcelExportService | 导出服务 |
| ExcelImportService | 导入服务 |

|------|------|----------|

### 数据验证规则

```python
# 示例：路口数据验证
def validate_intersection_row(row):
    errors = []
    if not row.get('name'):
        errors.append('路口名称不能为空')
    if row.get('type') not in ['十字路口', '丁字路口', '行人过街', '其他']:
        errors.append('无效的路口类型')
    return errors

def validate_project_row(row):
    errors = []
    if not row.get('name'):
        errors.append('项目名称不能为空')
    if not row.get('warranty_expire_date'):
        errors.append('质保到期日期不能为空')
    return errors
```

**部分失败响应**:

  "count": 0,
  "errors": ["第1行: 表头格式不正确"]

### 批量处理策略

```python
# 分批提交，避免大数据量事务超时
BATCH_SIZE = 100

def import_batch(items):
    for i in range(0, len(items), BATCH_SIZE):
        batch = items[i:i + BATCH_SIZE]
        db.session.bulk_insert_mappings(Model, batch)
        db.session.commit()
```

### 错误处理机制

| 错误类型 | 处理策略 |
|---------|----------|
| 数据格式错误 | 跳过该行，记录错误 |
| 必填字段为空 | 跳过该行，记录错误 |
| 外键关联不存在 | 跳过该行，记录错误 |
| 数据库写入失败 | 回滚事务，返回错误 |

### 事务管理

```python
def import_excel(file, import_type):
    try:
        # 验证文件
        if not file:
            raise ValueError("未选择文件")

        # 读取数据
        data = read_excel(file)

        # 批量导入
        for batch in chunked(data, BATCH_SIZE):
            process_batch(batch)

        db.session.commit()
        return {'count': len(data), 'errors': []}

    except Exception as e:
        db.session.rollback()
        raise e
```

## 替代方案考虑

| 替代方案 | 优点 | 缺点 |
|---------|------|------|
| CSV格式 | 文件小，兼容性好 | 格式简单，无样式 |
| JSON格式 | 结构清晰，层次分明 | 非标准Excel，用户不熟悉 |
| 数据库直连 | 数据量大时效率高 | 需要额外权限，安全性问题 |
| 第三方导入库 | 功能丰富 | 增加依赖，学习成本 |

## 后果

### 正面
- Excel用户熟悉，学习成本低
- 支持批量操作，提高效率
- 错误提示明确，便于用户修正
- 实现相对简单

### 负面
- 大数据量导入性能有限
- 需考虑编码问题（UTF-8）
- 复杂关系数据导入困难
- 无法处理附件等二进制数据

## 未来优化方向

1. **大文件分片上传**：支持超大文件导入
2. **异步导入**：后台处理，任务队列
3. **导入进度条**：实时显示导入进度
4. **模板可视化编辑**：在线设计导入模板
5. **数据映射**：灵活配置字段对应关系
